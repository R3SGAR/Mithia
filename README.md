**Environment**
- conda 24.11.1
- Flask 3.1.2
- Python 3.12.8

## Install
```bash
$ conda create -n mithiav1 python=3.12.8
$ conda activate mithiav1
$ pip install flask
$ git clone https://github.com/R3SGAR/Mithia.git
$ cd quiz_app
```

## Run
```bash
python app.py
```

*Go to http://127.0.0.1:5000*
